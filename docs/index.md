# Malleon Documentation

Set up your app, capture replays, analyze errors and requests, and create automated tests.

## Quick Start
1. Create an account and log in.
2. Create an app and copy the App ID.
3. Integrate the Replay SDK in your web app.
4. Upload source maps for readable stack traces.
5. Use the web app to find replays, errors, and performance issues.
6. Turn critical replays into automated tests.
